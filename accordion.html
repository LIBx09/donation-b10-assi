Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a structured representation of an HTML or XML document, allowing developers to manipulate the content and elements of a webpage using JavaScript.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, there are several methods: getElementById(), querySelector() etc.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple listeners on individual child elements. It is useful as it reduces memory usage, improves performance, and simplifies handling dynamically added elements.

How do you manipulate an element's attribute and styles using the DOM?

Manipulate an element's attributes using methods like setAttribute() and getAttribute(). For styles, use element.style to change inline styles or classList to add/remove CSS classes dynamically.
